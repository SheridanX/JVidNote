# JVidNote

一个基于 FFmpeg 和 Qt 的跨平台视频笔记工具。

## 依赖

- **FFmpeg** — 音视频解码与处理
- **Qt 6** — 图形用户界面（计划中）
- **CMake** ≥ 3.16
- **pkg-config**

## 构建

```bash
cd build
cmake ..
cmake --build .
```
